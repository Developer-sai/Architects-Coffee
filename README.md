# ☕ Architects Coffee | Specialty Coffee Studio

> **"Drafted with Accuracy. Brewed to Perfection."**

An artistic, high-performance web application designed for **Architects Coffee**—where architectural design precision meets single-origin artisanal coffee roasting.

---

## 🌟 Key Features & Highlights

### 📐 Architectural Design System
- **Obsidian Dark Aesthetic**: Deep slate palette (`#0A0C10`), warm metallic bronze/copper highlights (`#D4A373`), and translucent glassmorphism overlays (`rgba(18, 22, 30, 0.75)`).
- **Blueprint Grid Overlay**: Structural vector grid background with precision blueprint cyan highlights (`#48CAE4`).
- **Typography Suite**:
  - `Cinzel` (Header & Brand titles)
  - `Syne` (Subheadings & Geometric accents)
  - `Plus Jakarta Sans` (Body text)
  - `Fira Code` (Extraction ratios & technical specs)

---

### ☕ 1. The Blueprint Menu
- **Dynamic Search & Filtering**: Instant realtime filter by keyword, bean origin, or flavor notes.
- **Category Tabs**:
  - `All Specs`
  - `Structural Espressos`
  - `Filter & Pour-Overs`
  - `Cold Extractions`
  - `Artisanal Bakes`
  - `Single Origin Beans`
- **Technical Cards**: Showcases extraction ratios (e.g. `1:16`), elevation, processing method, and tasting tags.

---

### 🧪 2. "Draft Your Custom Brew" Studio
- **Real-Time Visual Coffee Schematic**: Interactive glass layer rendering that adjusts visually as you customize:
  - **Base Espresso**: Single Origin Ethiopia, Ristretto Guatemala, Cold Brew Draft, Kyoto Matcha.
  - **Extraction Shots**: Double (18g), Triple (24g), Single Ristretto.
  - **Milk / Texture**: Whole Velvet, Oat (Minor Figures), Almond Silk, Cold Foam Macadamia.
  - **Flavor Accents**: Smoked Vanilla, Dark Cocoa Nibs, Cardamom Bronze, Pure.
- **Live Price Calculation**: Dynamic price updating based on selected ingredients.

---

### 📐 3. Studio Workspace Desk Booking
- **Interactive Floorplan Grid**: Real-time visual seat map showing available, selected, and reserved desks:
  - Drafting Desks (Sunlight Window Bar & Main Hall)
  - Focus Pods (Quiet Acoustic Booths)
  - Team Conference Desks (4-Person Desks)
- **Booking Reservation Form**: Date, time slot selection, guest count, and instant confirmation toast.

---

### 🛍️ 4. Slide-Out Blueprint Cart Drawer
- **Itemized Cart Management**: Quantity increments/decrements and removal.
- **Promo Code Engine**: Enter `ARCH20` to unlock **20% Architectural Discount**.
- **Tax & Total Calculator**: Live NY tax rate (8.875%) calculation and checkout confirmation.

## 🚀 Quick Setup & Local Execution

### Option A: Python Built-in Server
```bash
# Navigate to project folder
cd "Architects Coffee"

# Start HTTP Server on port 8080
python -m http.server 8080
```
Open your browser to: **`http://localhost:8080`**

### Option B: Node.js Serve / Live Server
```bash
npx serve .
```
